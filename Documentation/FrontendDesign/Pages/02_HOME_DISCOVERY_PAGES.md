## 9. Shared Components

### 9.1 StoryCard

```tsx
interface StoryCardProps {
  story: Story;
  variant?: 'default' | 'compact' | 'list';
  showMatch?: boolean; // For recommendations
  showRank?: boolean;  // For rankings
}

<StoryCard story={story}>
  <StoryCover src={story.coverUrl} alt={story.title} />
  <StoryInfo>
    <StoryTitle>{story.title}</StoryTitle>
    <StoryAuthor>{story.author.name}</StoryAuthor>
    <StoryMeta>
      <Rating value={story.rating} />
      <ReadCount count={story.readCount} />
    </StoryMeta>
    {showMatch && <MatchBadge percent={story.matchPercent} />}
  </StoryInfo>
</StoryCard>
```

## 10. Performance Optimization

### 10.1 Data Fetching Strategy

| Section | Strategy | Stale Time |
|---------|----------|------------|
| Hero | Prefetch on landing | 1 hour |
| Featured | Prefetch | 30 min |
| Trending | Background refetch | 5 min |
| Recommendations | User-triggered | 30 min |
| Search results | On-demand | 1 min |

### 10.2 Image Optimization

```tsx
<StoryCover
  src={story.coverUrl}
  alt={story.title}
  sizes="(max-width: 640px) 150px, 200px"
  loading="lazy"
  placeholder="blur"
  blurDataURL={story.coverBlurHash}
/>
```

### 10.3 Virtual Scrolling

```tsx
// For large lists (100+ items)
<VirtualList
  height={600}
  itemCount={stories.length}
  itemSize={120}
  renderItem={({ index, style }) => (
    <div style={style}>
      <StoryCard story={stories[index]} variant="list" />
    </div>
  )}
/>
```
