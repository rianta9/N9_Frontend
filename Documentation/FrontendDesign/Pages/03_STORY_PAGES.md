### 2.3 Related Backend APIs

#### Story Detail Endpoints
| Endpoint | Method | Purpose |
|----------|--------|----------|
| `/api/v1/stories/{id}` | GET | Story details |
| `/api/v1/stories/{id}/chapters` | GET | Chapter list with pagination |
| `/api/v1/stories/{id}/chapters/{chapterId}` | GET | Single chapter metadata |
| `/api/v1/stories/{id}/statistics` | GET | Story statistics |

#### Interaction Endpoints
| Endpoint | Method | Purpose |
|----------|--------|----------|
| `/api/v1/stories/{id}/reviews` | GET | Story reviews (paginated) |
| `/api/v1/stories/{id}/reviews` | POST | Submit new review |
| `/api/v1/stories/{id}/reviews/{reviewId}` | PUT | Update review |
| `/api/v1/stories/{id}/reviews/{reviewId}` | DELETE | Delete review |
| `/api/v1/stories/{id}/rating` | GET | Get user's rating |
| `/api/v1/stories/{id}/rating` | POST | Rate story |
| `/api/v1/stories/{id}/rating` | DELETE | Remove rating |

#### Library Endpoints
| Endpoint | Method | Purpose |
|----------|--------|----------|
| `/api/v1/library/add` | POST | Add story to library |
| `/api/v1/library/remove/{storyId}` | DELETE | Remove from library |
| `/api/v1/library/check/{storyId}` | GET | Check if in library |

#### Social Endpoints
| Endpoint | Method | Purpose |
|----------|--------|----------|
| `/api/v1/stories/{id}/share` | POST | Generate share link |
| `/api/v1/stories/{id}/similar` | GET | Similar stories |
| `/api/v1/stories/{id}/report` | POST | Report story |

## 10. Interactions

### 10.1 Story Actions

| Action | Auth Required | Trigger | Result |
|--------|---------------|---------|--------|
| Start Reading | No | Click button | Navigate to first unread chapter |
| Continue Reading | No | Click button | Navigate to last read position |
| Add to Library | Yes | Click button | Open library modal |
| Rate | Yes | Click star | Open rating modal |
| Bookmark | Yes | Click icon | Toggle bookmark |
| Share | No | Click icon | Open share modal |
| Report | Yes | Click icon | Open report modal |
| Follow Author | Yes | Click button | Toggle follow |

### 10.2 Chapter Actions

| Action | Auth Required | Trigger | Result |
|--------|---------------|---------|--------|
| Read Chapter | No (free) / Yes (premium) | Click row | Navigate to reader |
| Unlock Chapter | Yes | Click locked chapter | Open purchase modal |
| Mark as Read | Yes | Swipe right (mobile) | Update read status |

## 12. Performance Optimizations

### 12.1 Data Prefetching

```typescript
// Prefetch story on hover
function StoryLink({ slug, children }) {
  const queryClient = useQueryClient();
  
  const prefetch = () => {
    queryClient.prefetchQuery({
      queryKey: ['story', slug],
      queryFn: () => api.stories.getBySlug(slug),
      staleTime: 5 * 60 * 1000,
    });
  };
  
  return (
    <Link to={`/story/${slug}`} onMouseEnter={prefetch}>
      {children}
    </Link>
  );
}
```

### 12.2 Chapter List Virtualization

```tsx
// For stories with 500+ chapters

  chapters={chapters}

  overscan={5}
```
